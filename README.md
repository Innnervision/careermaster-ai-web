# CareerMaster AI Web

CareerMaster AI is an AI-powered career management application designed to help users navigate their professional development and career growth.

## Live Demo

Visit the live application: [https://innnervision.github.io/careermaster-ai-web/](https://innnervision.github.io/careermaster-ai-web/)

## Features

- AI-powered career guidance
- Skill assessment and development
- Job market insights
- Resume and portfolio optimization
- Career path planning

- Built with Flutter
- Web deployment via GitHub Pages
